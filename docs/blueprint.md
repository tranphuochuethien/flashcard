# **App Name**: Nihongo Flash

## Core Features:

- Vocabulary Entry: Allow users to input new Japanese vocabulary, including Kanji, Hiragana, Han Viet, Vietnamese meaning, and IT context notes. 
- Batch Import: Enable admin users to import vocabulary entries in bulk via CSV upload to the Cloud SQL database. 
- Flashcard Display: Present vocabulary as flashcards, showing either the Japanese or Vietnamese side first. 
- Flip Action: Allow users to flip the flashcard to reveal the other side (Japanese/Vietnamese).
- Random shuffle: Shuffle and go to the next flashcard in a shuffled order
- AI Note Enhancement Tool: Generative AI tool that suggests IT context notes or expands upon existing notes for a given vocabulary entry to improve understanding and recall, before adding into the database.
- Admin Panel: Backend page (protected) for easy CRUD (create, read, update, delete) operations.
